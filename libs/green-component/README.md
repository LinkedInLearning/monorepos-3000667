# @libs/rainbow

> a rainbow lib

[![NPM](https://img.shields.io/npm/v/@libs/rainbow.svg)](https://www.npmjs.com/package/@libs/rainbow) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save @libs/rainbow
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from '@libs/rainbow'
import '@libs/rainbow/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
